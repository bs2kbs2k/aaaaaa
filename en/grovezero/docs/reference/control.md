# Control

Program controls and events.

```cards
control.millis()
control.runInBackground(function() { })
control.reset()
control.waitMicros(4)
```

## Advanced

```cards
control.raiseEvent(0, 0)
control.onEvent(0, 0, function() { })
control.waitForEvent(0, 0)
control.panic(0)
control.deviceSerialNumber()
```
