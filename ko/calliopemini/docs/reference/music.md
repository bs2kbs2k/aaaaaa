# 소리

`P0` 핀을 통해 소리(음)를 출력합니다.

```cards
music.playTone(0, 0);
music.ringTone(0);
music.rest(0);
music.beat(BeatFraction.Whole);
music.tempo();
music.changeTempoBy(20);
music.setTempo(120);
```
