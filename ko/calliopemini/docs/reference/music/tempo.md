# 현재 빠르기(bpm)

현재 빠르기(bpm) 를 알아냅니다. (음악이 출력되는 속도).

```sig
music.tempo()
```

### 리턴값

* a [number](/reference/types/number) that means the beats per minute (number of beats in a minute of the music that the @boardname@ is playing).
