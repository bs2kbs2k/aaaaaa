# 빠르기(bpm) 설정

빠르기(bpm) (음악이 연주되는 속도) 를 원하는 속도로 설정 합니다.

```sig
music.setTempo(60)
```

## 시뮬레이터

이 함수는 @boardname@ 와 일부 웹브라우저에서만 정상적으로 동작합니다.

## 매개 변수

* `빠르기(bpm)`. 빠르기로 설정할 bpm(beats per minute) [수(정수)](/types/number). (bmp 은 @boardname@ 보드를 이용해 소리를 출력할 때, 분당 비트 수를 의미합니다.)
