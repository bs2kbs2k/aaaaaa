# 현재 빠르기(bpm)

현재 빠르기(bpm) 를 알아냅니다. (음악이 출력되는 속도).

```sig
music.tempo()
```

## 리턴값

* [빠르기(bpm)](/types/number). 빠르기 bpm(beats per minute). (bmp 은 @boardname@ 보드를 이용해 소리를 출력할 때, 분당 비트 수를 의미합니다.)
