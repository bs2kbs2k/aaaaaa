# 대입 연산자

어떤 [변수](/blocks/variables/var)에 [값](/types/number)이나 [문자열](/types/string)을 저장 시키려면, 등호(=) 기호를 사용하세요.

어떤 값을 저장시키기 위해 등호 기호를 사용하면, 그 기호를 *대입 연산자*라고 부르고, 그 변수에 저장 시킨 것을 <0>값<0>이라고 부릅니다.</p> 

## ~hint

대입 연산자는, 원하는 모든 [데이터형](/types)의 변수들에 사용할 수 있습니다. *데이터형</0>은 저장하는 데이터의 종류를 의미하며, 수나 문자열과 같은 데이터의 종류를 의미합니다.</p>
