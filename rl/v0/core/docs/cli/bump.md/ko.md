# pxt-bump Manual Page

### @description 현재 버전 번호와 충돌합니다.

Bumps the version number of the package.

    pxt bump [--update] [--upload] 
    

## Options

### --update

Updates the pxt-core dependency version to the latest tag.

### --upload

Uploads the target upon bumping
